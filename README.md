# Network Scanner Tool

A Python-based mini network scanner inspired by Nmap. It scans a target IP address or hostname, detects open ports, tries to identify common services, and saves scan results to a text file.

## Features

- Scan a target IP address or hostname
- Detect open ports from a selected range
- Identify common services using known port mappings
- Save scan results to a file
- Beginner-friendly cybersecurity portfolio project

## Tech Stack

- Python 3
- socket
- scapy
